Module: TimelineView

TimelineView

new require("TimelineView")(opts)

TimelineView is a view that uses streamhub data to populate a horizontail timeline. It can optionally operate on just a given date range, embed meta content in a specified child element of the item's bodyHtml, and can be passed an onClick event handler.

Methods

<protected> _insertItem(item, col) → {Object}

Inserts a new piece of content into the dom. Used as a callback handler for collection.add events. If a metaElement was specified on creation, the element's contents will be parsed as json, and "eventType" and "eventImportant" will be parsed out and placed as data attributes on the event dot, useful for styling.

Parameters:

A piece of content that was streamed to this view from Streamhub.
